## 技术实现要点

### 1. 间隔重复算法（艾宾浩斯遗忘曲线）

```python
def get_next_review_interval(status, review_count):
    intervals = [1, 2, 4, 7, 15, 30, 60]
    if status == 'mastered':
        index = min(review_count, len(intervals) - 1)
        return timedelta(days=intervals[index])
    else:
        return timedelta(days=1)
```

### 2. 用户数据隔离

- 数据库表添加 `user_id` 字段
- API通过JWT获取当前用户ID
- 所有查询按 `user_id` 过滤

### 3. 学习模式实现

- **普通模式**：显示单词，点击显示答案，标记已掌握/学习中
- **拼写测试**：显示释义，用户输入单词拼写
- **选择题**：随机生成4个选项，包含正确答案
- **听音辨词**：使用Web Speech API播放发音，选择正确单词
